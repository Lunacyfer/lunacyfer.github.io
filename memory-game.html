Memory - Maiden Voyage

First game I ever worked on Memory - Maiden Voyage is a tiny indie game born after the announcement of the first Apple Watch in 2015. Back then, my friend Michael (iOS developer) and I decided to create a simple yet entertaining game with fast gameplay designed for users with an Apple Watch, especially for those who have some short spare time, like morning commuters. Memory - Maiden Voyage was, in fact, the first game playable from an Apple Watch and is still promoted by Apple in the watch category to this day.

Memory - Maiden Voyage is a reinterpretation of the classic memory game but with a more modern and abstract concept, inspired by two of my favorite games of the time: Monument Valley and Alto's Adventure. Set in space, it features a captivating geometric design of shapes that play with colours to create mesmerizing illusions.
